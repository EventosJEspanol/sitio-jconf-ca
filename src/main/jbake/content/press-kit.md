## CFP Invitation (Spanish)

Los grupos de usuarios Java de Guatemala, El Salvador, Nicaragua y Panamá de manera muy atenta los invitan a participar como speakers en el primer JConf Centroamérica el día 5 de diciembre de 2020.

### ¿Que es JConf?

Las JConf son conferencias independientes organizadas por los grupos de usuarios Java de diversos países, con el objetivo de generar puntos de encuentro entre comunidades alrededor de la Java Virtual Machine incluyendo Java, Kotlin, Scala, Groovy, Clojure, Android, Data Engineering, GraalVM y derivados.

En esta oportunidad los grupos de usuarios de la región nos hemos unido para continuar siendo el evento JVM más grande de Ámerica Central.

### ¿Como puedo participar?

Puedes apoyar a JConf de dos formas:

* Participando como ponente y registrando tu propuesta de ponencia (conferencia, Q&A y workshop) a traves de nuestra plataforma.

[https://jconf.org/cfp.html](https://jconf.org/cfp.html)

Los tópicos de la conferencia son:

    * Java Platform
    * JVM Languages
    * Server Side Java
    * Client Side Java
    * Embedded Java
    * Big (Data with) Java
    * Tools, architecture and strategies
    * Startups with Java

* Compartiendo el evento con tus comunidades utilizando los elementos disponibles en nuestro [press-kit](https://jconf.org/press-kit.html)

### Fechas importantes:

Inicio CFP: 2020/10/05
Cierre CFP: 2020/11/06
Notificación de aceptación de propuestas: 2020/11/09

¡Los desarrolladores de Centroamérica esperan por ti!

## CFP Invitation (English)

The Java user groups from Guatemala, El Salvador, Nicaragua and Panama very attentively invite you to participate as speakers in the first JConf Central America on December 5, 2020.

### What is JConf?

The JConf are independent conferences organized by Java user groups from various countries, with the aim of generating collaboration spaces between communities around the Java Virtual Machine including Java, Kotlin, Scala, Groovy, Clojure, Android, Data Engineering, GraalVM and derivatives.

This time the Java user groups of the region have come together to promote the largest JVM event in Central America.

### How can I participate?

You can support JConf in two ways:

* Participating as a speaker and registering your presentation proposal (conference, Q&A and workshop) through our platform.

[https://jconf.org/cfp.html](https://jconf.org/cfp.html)

The topics of the conference are:

    * Java Platform
    * JVM Languages
    * Server Side Java
    * Client Side Java
    * Embedded Java
    * Big (Data with) Java
    * Tools, architecture and strategies
    * Startups with Java

* Sharing the event with your communities using the elements available in our [press-kit](https://jconf.org/press-kit.html)

### Important dates:

CFP start: 2020/10/05
CFP closing: 2020/11/06
Notification of acceptance of proposals: 2020/11/09

Central American developers are waiting for you!
