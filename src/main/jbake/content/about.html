title=About date=2018-11-10 type=page status=published ~~~~~~
JConf Centroamérica es un evento anual dirigido a crear un centro de actividad para Java leaders, entusiastas y desarrolladores de América Central y el mundo :).

El evento es promovido por Grupos de Usuarios de Java (JUGs) de Centroamérica, quienes sostienen encuentros mensuales abiertos para todos, generando espacios de discusión en temas relacionados con Java (como JVM). El principal objetivo del evento es promover Java como plataforma al promover el conocimiento abierto como otros espacios ya existentes en ciudades alrededor del mundo.

JConf Centroamérica nace de la anterior Conferencia de Java en Centroamérica (JConf Guatemala, desde 2011) , siendo el evento más exitoso de Java en la región (hasta donde sabemos :) ). El evento presenta charlas y talleres en tres/cuatro tracks, permitiendo a los asistentes tener acceso libre/gratuito a este conocimiento.

El evento es el resultado de constantes encuentros entre miembros JUG, quedando abierto a todo el mundo.
